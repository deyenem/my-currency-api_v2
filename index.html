API Documentation

Everything you need to integrate currency conversion into your application

GET /api/convert

Currency Conversion

Convert any amount from one currency to another with real-time exchange rates and lightning-fast response times.

ParameterTypeRequiredDescription
fromstringRequiredSource currency code (e.g., USD)
tostringRequiredTarget currency code (e.g., EUR)
amountnumberOptionalAmount to convert (default: 1)
formatstringOptionalResponse format: json or xml
{ "success": true, "from": "USD", "to": "EUR", "rate": 0.8678, "amount": 100, "converted": 86.78, "timestamp": "2025-06-21T15:30:00.000Z", "data_source": "local_storage", "cache_status": "fresh" }
GET /api/rates

All Exchange Rates

Get all available exchange rates in a single request. Perfect for populating currency selectors or building rate comparison tools.

ParameterTypeRequiredDescription
basestringOptionalBase currency (currently USD only)
formatstringOptionalResponse format: json or xml
{ "success": true, "base_code": "USD", "conversion_rates": { "EUR": 0.8678, "GBP": 0.7427, "JPY": 145.8389 }, "total_currencies": 170 }
GET /api/currencies

Currency List

Get a complete list of supported currencies with their full names, optional rates, and search functionality.

ParameterTypeRequiredDescription
searchstringOptionalFilter currencies by code or name
include_ratesbooleanOptionalInclude exchange rates (default: false)
sortstringOptionalSort by "code" or "name"
{ "success": true, "currencies": [ { "code": "USD", "name": "United States Dollar" } ], "total": 170 }
POST /api/bulk

Bulk Conversion

Convert multiple currency pairs in a single request. Supports up to 200 conversions with lightning-fast processing.

// Request POST /api/bulk { "conversions": [ {"from": "USD", "to": "EUR", "amount": 100} ] } // Response { "success": true, "total_conversions": 1, "results": [...] }
GET /api/health

Health Check

Monitor API health, performance metrics, system status, and data freshness for reliable service monitoring.

{ "status": "healthy", "response_time_ms": 23, "data_available": true, "total_currencies": 170 }
